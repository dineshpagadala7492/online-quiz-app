# 🧠 Online Quiz App

A web-based quiz platform built using Python Flask and SQLite.

## 🛠️ Tech Stack
- Python  
- Flask  
- HTML, CSS, JavaScript  
- SQLite  

## ⚙️ Features
✅ User login and score tracking  
✅ Dynamic question loading from database  
✅ Real-time result calculation  
✅ Simple, mobile-friendly UI
